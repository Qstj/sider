# Sider
A novel deep learning-based drug-side effect frequency prediction model.

# Requirements

* python == 3.6.13
* pytorch == 1.10.2
* torch-geometric 2.0.3
* numpy == 1.19.5
* scipy == 1.5.4
* scikit-learn == 0.24.2
* networkx == 2.5.1
* rdkit == 2020.09.1.0

# Run
```bash
python main.py --split drug --drug_boost
```
